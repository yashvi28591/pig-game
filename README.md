# 🎲 Pig Game

A fun two-player dice game built using **HTML**, **CSS**, and **JavaScript**. Players take turns rolling a dice and collecting points. The first player to reach the target score wins the game.

## 🚀 Features

- Two-player gameplay
- Random dice roll generation
- Current score tracking
- Total score accumulation
- Hold functionality to save points
- Automatic player switching
- New Game reset button
- Responsive and interactive UI

## Game Rules

1. Roll the dice.
2. If you roll a 1, your current score becomes 0 and the turn switches.
3. If you roll 2–6, the value is added to your current score.
4. Click Hold to add the current score to your total score.
5. The first player to reach 100 points wins.
